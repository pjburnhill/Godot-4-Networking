# Godot-4-Networking

Basic Networking Starter

## Basics included

- Set up server
- Load level
- Spawn players

## Synchronization

- Sync `player:player_id` (at spawn) and `player:position`
- Sync input direction (float)
- Use rpc local call for jump

- Move players (local & puppet) based on synchronized input
